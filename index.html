Data Collection Methodology

How We Built This Dataset

01

Housing.com — Web Scraping

Transaction-level data: listing price, flat size, amenity count, project details. Filtered to developer-sold only (no resale), new bookings (age <1 year), 3BHK configuration — ensuring sample homogeneity.

02

MagicBricks — Location Intelligence

Location-level signals: average rate (₹/sqft), 1-year price trend (%), locality rank (demand proxy), active supply count. Enriches each listing with neighbourhood context not visible in transaction data alone.

03

Gujarat RERA Portal — Compliance Data

Quarterly compliance scores, RERA registration IDs, project cost estimates, and loan exposure ratios. Critical transparency layer — often inaccessible to regular buyers buried in government portals.
